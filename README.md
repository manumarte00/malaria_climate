### Effect of Climatic and Environmental factors on Malaria

-----------------------------------------------------------------------------

This study investiagtes the effect of climatic and environmental factors on Malaria using a mixed effects logistic regression combined with a Distributed Lag Non-Linear Model (DLNM). This repository contains the original analysis code, along with simulated data designed to replicate the analytical approach of the study.

IMPORTANT! The manuscript is currently under review. Results are subject to change.

-----------------------------------------------------------------------------

This repository contains:

- *main_analysis.R* contains the overall effect of the climatic variables
- *effects_lags.R* includes the effects along the lags
- *r_squared.R* contains the main effect of the socio-economic covariates included in the analysis
- The files ending with *int.R* includes the effect modification analysis for each climatic variable
- *control_lags.R* and *control_micro.R* contain the sensitivity analyses reported in the manuscript
- *Data_simulation_malaria.qmd* and *Data_simulation_malaria.html* contains the demo of the original analysis with a simulation of the data
